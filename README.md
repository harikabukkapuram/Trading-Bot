# Trading Bot Template

This is a starter template. Replace placeholder code with the implementation or ask ChatGPT to generate each file.
Add your Binance Testnet API keys in a .env file.

Run:
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
python cli.py --symbol BTCUSDT --side BUY --type MARKET --quantity 0.001
