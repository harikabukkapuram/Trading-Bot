# Validation Tests

These help demonstrate your input validation and error handling.

## 5. Invalid Side

```bash
python cli.py --symbol BTCUSDT --side HOLD --type MARKET --quantity 0.001
```

Expected:

```
Validation Error: Side must be BUY or SELL.
```

---

## 6. Invalid Order Type

```bash
python cli.py --symbol BTCUSDT --side BUY --type STOP --quantity 0.001
```

Expected:

```
Validation Error: Order type must be MARKET or LIMIT.
```

---

## 7. Missing Price for LIMIT

```bash
python cli.py --symbol BTCUSDT --side BUY --type LIMIT --quantity 0.001
```

Expected:

```
--price is required for LIMIT orders.
```

## 8. Invalid Quantity

```bash
python cli.py --symbol BTCUSDT --side BUY --type MARKET --quantity -1
```

Expected:

```
Validation Error: Quantity must be greater than zero.
```
